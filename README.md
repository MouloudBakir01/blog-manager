## üöÄ Project Description

**Blog Manager** is a frontend-only application developed for the Code2Work project.  
It allows students and young professionals to manage and organize their personal articles without any backend or external API.

### ‚ú® Key Features
- Create, read, update, and delete articles (CRUD)
- Store data **locally** with `localStorage` (no backend)
- Manage articles by **category, title, or creation date**
- Edit articles via a **modal dialog**
- Confirm deletion with user validation
- Filter and sort articles by multiple criteria
- Fully typed with **TypeScript**
- Responsive design and accessible UI (ARIA attributes)
- Navigation between pages using **React Router**
